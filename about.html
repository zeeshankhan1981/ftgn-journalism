About Against The Tide

  • Project Overview

    Against The Tide is a collection of 250-word essays documenting civilian experiences during U.S. military bombardments. This project provides fact-based, compassionate journalism that focuses on the human impact of warfare.

  • Methodology

    Essays are based on verified historical accounts, eyewitness testimonies, and scholarly sources. The 250-word format creates focused, impactful narratives that can be easily read and shared.

  • Author

    TorchScribe is a collective of journalists and researchers specializing in the documentation of civilian experiences in conflict zones. Their work brings attention to overlooked aspects of military conflicts.

  • Technical Implementation

    This website features a typewriter-inspired design that reflects the journalistic nature of the project. All essays are stored as Markdown files, which can be easily edited with Emacs Doom.

  • Keyboard Shortcuts

    Alt+S: Focus search box
    Alt+A: Go to About page
    Alt+Home: Go to Home page
